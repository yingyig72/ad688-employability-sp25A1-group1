Skill Gap Analysis

Overview

This section compares the skills required in IT job postings against the actual skills of our team members. By identifying gaps between what the market demands and what we currently possess, we can develop targeted learning plans.

2. Industry Skill Requirements

Extract the most in-demand skills from the job postings dataset to understand what employers are looking for.

Extract skill demand from job descriptions 
import re
from collections import Counter

# Skills to search for in job descriptions
target_skills = list(df_skills.columns)
additional_skills = ["Kubernetes", "Azure", "GCP", "Agile", "Scrum",
                     "TensorFlow", "PyTorch", "Power BI", "Linux",
                     "MongoDB", "PostgreSQL", "ETL", "Airflow", "Kafka",
                     "Scikit-learn", "Data Pipeline", "React", "Node.js"]
all_skills = target_skills + additional_skills

# Find description column
desc_col = None
for col in df.columns:
    if any(kw in col.lower() for kw in ['description', 'skills', 'requirement', 'qualification', 'detail']):
        desc_col = col
        break

if desc_col:
    text_data = df[desc_col].dropna().str.lower()
    total_postings = len(text_data)

    skill_demand = {}
    for skill in all_skills:
        count = text_data.str.contains(re.escape(skill.lower()), case=False, na=False).sum()
        if count > 0:
            skill_demand[skill] = round(count / total_postings * 100, 1)

    demand_df = pd.DataFrame(
        sorted(skill_demand.items(), key=lambda x: x[1], reverse=True),
        columns=["Skill", "Demand (% of postings)"]
    )

    fig = px.bar(
        demand_df.head(20),
        x="Demand (% of postings)", y="Skill",
        orientation='h',
        title="Top 20 Skills — % of Job Postings Mentioning Each",
        color="Demand (% of postings)",
        color_continuous_scale="Blues"
    )
    fig.update_layout(yaxis=dict(autorange="reversed"), coloraxis_showscale=False, height=550)
    fig.show()
else:
    # Fallback: use a manually curated demand list
    print("Description column not found — using curated skill demand estimates.")
    skill_demand = {
        "Python": 72, "SQL": 65, "AWS": 48, "Machine Learning": 42,
        "Cloud Computing": 38, "Docker": 32, "Excel": 30, "Tableau": 28,
        "Git": 26, "Communication": 25, "R": 20, "JavaScript": 18,
        "Spark": 15, "Deep Learning": 14, "NLP": 12
    }
    demand_df = pd.DataFrame(
        sorted(skill_demand.items(), key=lambda x: x[1], reverse=True),
        columns=["Skill", "Demand (% of postings)"]

3. Gap Analysis: Team vs. Market

Compare team skills to industry demand 
# Normalize team skills: average proficiency on 0-100 scale
team_avg = df_skills.mean()
team_avg_pct = (team_avg / 5 * 100).round(1)

# Build comparison DataFrame for overlapping skills
overlap_skills = [s for s in team_avg_pct.index if s in skill_demand]

comparison = pd.DataFrame({
    "Skill": overlap_skills,
    "Team Proficiency (%)": [team_avg_pct[s] for s in overlap_skills],
    "Market Demand (%)":    [skill_demand[s] for s in overlap_skills]
})
comparison["Gap"] = comparison["Market Demand (%)"] - comparison["Team Proficiency (%)"]
comparison = comparison.sort_values("Gap", ascending=False)

5. Improvement Plan

Based on the gap analysis, here is a structured plan for the team:

Recommended improvement plan 
plan = {
    "Skill": [],
    "Gap Severity": [],
    "Recommended Resources": [],
    "Team Members to Focus": []
}

for _, row in comparison.sort_values("Gap", ascending=False).iterrows():
    skill = row["Skill"]
    gap = row["Gap"]
    if gap <= 0:
        continue

    # Who on the team is weakest at this skill?
    member_scores = df_skills[skill] if skill in df_skills.columns else pd.Series()
    weak_members = member_scores[member_scores <= 2].index.tolist() if len(member_scores) > 0 else ["All"]

    # Suggest resources based on skill
    resources = {
        "Python": "Coursera – Python for Everybody; LeetCode practice",
        "SQL": "Mode Analytics SQL Tutorial; HackerRank SQL",
        "Machine Learning": "Andrew Ng's ML Specialization (Coursera)",
        "Cloud Computing": "AWS Cloud Practitioner cert; Azure Fundamentals",
        "AWS": "AWS Cloud Practitioner / Solutions Architect prep",
        "Docker": "Docker official docs; Play with Docker labs",
        "Tableau": "Tableau Public tutorials; Coursera Tableau Specialization",
        "Git": "GitHub Skills interactive courses",
        "R": "R for Data Science (Hadley Wickham) – free online",
        "Spark": "Databricks Academy free courses",
        "Deep Learning": "fast.ai Practical Deep Learning; deeplearning.ai",
        "NLP": "Hugging Face NLP course (free)",
        "JavaScript": "freeCodeCamp JavaScript curriculum",
        "Communication": "Toastmasters; business writing workshops",
        "Excel": "Excel Skills for Business (Coursera)",
    }

    plan["Skill"].append(skill)
    plan["Gap Severity"].append(f"{gap:+.1f}")
    plan["Recommended Resources"].append(resources.get(skill, "Self-study & online tutorials"))
    plan["Team Members to Focus"].append(", ".join(weak_members) if weak_members else "—")

plan_df = pd.DataFrame(plan)

Peer teaching opportunities

Key Takeaways

  • The largest gaps tend to be in cloud technologies (AWS, Docker, Cloud Computing) and advanced analytics (Deep Learning, NLP, Spark).
  • Core skills like Python, SQL, and Excel are relatively strong across the team.
  • A peer mentoring system can accelerate learning — team members with advanced skills in one area can coach others.
  • Each member has a personalized priority list to guide their professional development over the coming months.
